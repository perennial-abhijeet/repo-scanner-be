## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Environment variables

```bash

NODE_ENV=
PORT=
LOG_LEVEL=
```

## Environment variables Description

```bash

NODE_ENV= (Set Node environment)
PORT= (Set Port on which application need to be run)
LOG_LEVEL= (Log level for application)
```
